## **🔗 GitHub Integration**

### **What It Does**
- Students push code to GitHub repos
- Faculty reviews code on GitHub
- Submissions tracked in real-time
- Feedback posted in system

### **Setup Steps** (Optional)
1. Create GitHub OAuth App (get Client ID)
2. Create Personal Access Token (GitHub scope)
3. Create GitHub organization for classroom
4. Configure in InternTrack Settings
5. Students link their repos

### **Student Workflow**
```
1. Student pushes code to GitHub
2. Login to InternTrack
3. Click "Submit Work"
4. Paste GitHub repo link
5. Add notes
6. Click Submit
7. Faculty reviews on GitHub
8. Faculty posts feedback in system
9. Student sees feedback in dashboard
```
